# 🧠 RouterMind

> **Voice-driven AI model selector** — 语音驱动的 AI 模型智能选择器

RouterMind 是一款通过自然语音交互帮助你从 OpenRouter 200+ 模型中快速找到最优 LLM 的 iOS 应用。只需说出你的需求，AI 引擎智能分析并推荐最适合你的模型。

RouterMind is a voice-driven AI model selector that helps you find the optimal LLM model on OpenRouter through natural voice interaction. Simply speak your requirements and get intelligent recommendations.

---

## ✨ Features / 功能

### 🎤 Voice-First Interaction / 语音优先交互
Just speak naturally — "I need a free coding model" or "找一款适合创意写作的模型" — and RouterMind handles the rest.

### 📊 Intelligent Model Scoring / 智能模型评分
| Dimension / 维度 | Description |
|:--|:--|
| Quality / 质量 | Output quality & accuracy |
| Speed / 速度 | Response latency |
| Cost / 成本 | Token pricing efficiency |
| Context / 上下文 | Context window size |
| Capability / 能力 | Task-specific strengths |
| Reliability / 可靠 | Uptime & consistency |

### 🏆 Triple Recommendation / 三重推荐
| Label / 标签 | Meaning / 含义 |
|:--|:--|
| 🥇 **Champion** | Best value recommendation — 最佳性价比推荐 |
| 🌿 **Eco-Choice** | Cheapest model scoring 70%+ quality — 最便宜且质量 70+ |
| ⚡ **Absolute Power** | Most capable model available — 最强性能之选 |

### 🔧 Additional Capabilities
- **Browse 200+ models** with type filters (Chat, Image, Video, Embedding, TTS, STT)
- **Bilingual support** — Chinese & English / 中英双语支持
- **Favorites management** — save preferred models for quick access
- **Markdown-rendered AI recommendations** with streaming output
- **Real-time transcription** for voice input

- **Language:** Swift / SwiftUI

- **API:** OpenRouter API (200+ models)
