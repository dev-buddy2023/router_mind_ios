## English

### Getting Started

RouterMind helps you find the best AI model on OpenRouter through natural voice or text input.

1. **Enter your API Key** — Go to Settings and paste your [OpenRouter API Key](https://openrouter.ai/keys)
2. **Speak or type** — Tap the microphone button and say what you need, e.g. _"Find me a cheap model for coding"_ or _"我需要一个擅长创意写作的模型"_
3. **Get recommendations** — RouterMind scores 200+ models across 6 dimensions and presents the top 3 picks

### The Three Recommendations

| Label | What It Means |
|:--|:--|
| 🥇 **Champion** | Best overall value — quality meets affordability |
| 🌿 **Eco-Choice** | Cheapest model scoring ≥70% quality |
| ⚡ **Absolute Power** | Most capable model regardless of cost |

### FAQ

#### Do I need my own API key?
**Yes.** RouterMind uses your personal OpenRouter API key to fetch models and generate recommendations. You can get a free key at [openrouter.ai/keys](https://openrouter.ai/keys).

#### Is my voice data stored anywhere?
**No.** Voice is transcribed entirely on your device via iOS Speech Recognition. Raw audio is never recorded, stored, or sent anywhere.

#### Which models are supported?
RouterMind covers **200+ models** on OpenRouter, including GPT-4o, Claude 3.5, Gemini, Llama 3, DeepSeek, Mistral, and more. Filters support Chat, Image, Video, Embedding, TTS, and STT model types.

#### How is the model score calculated?
Each model is scored across **6 dimensions**: Quality, Speed, Cost, Context Window, Capability (task fit), and Reliability. The scores are based on OpenRouter's public data and model specs.

#### Does RouterMind work offline?
No. An internet connection is required to communicate with the OpenRouter API.

#### Can I use RouterMind in Chinese?
**Yes.** RouterMind fully supports both English and Chinese — including voice input, UI, and AI recommendations.

### Troubleshooting

**"Invalid API Key" error**
- Make sure your key is entered correctly in Settings
- Verify the key is active on [openrouter.ai/keys](https://openrouter.ai/keys)
- Ensure your OpenRouter account has sufficient credits

**No models returned**
- Check your internet connection
- OpenRouter may be experiencing temporary downtime
- Try refreshing the model list in Settings

**Voice input not working**
- Grant microphone permission in iOS Settings → RouterMind
- Check that your device is not in Silent mode
- Ensure a stable internet connection

**App crashes or stalls**
- Close the app and reopen
- Update to the latest version from the App Store
- If the issue persists, contact us

### 常见问题

#### 需要自己的 API Key 吗？
**需要。** RouterMind 使用您个人的 OpenRouter API Key 来获取模型列表并生成推荐。可在 [openrouter.ai/keys](https://openrouter.ai/keys) 免费获取。

#### 语音数据会被存储吗？
**不会。** 语音完全在设备本地通过 iOS 语音识别转写。原始音频不会被记录、存储或发送。

#### 支持哪些模型？
RouterMind 覆盖 OpenRouter 上 **200+ 模型**，包括 GPT-4o、Claude 3.5、Gemini、Llama 3、DeepSeek、Mistral 等。支持按 Chat、Image、Video、Embedding、TTS、STT 类型筛选。

#### 模型评分怎么算的？
每个模型从 **6 个维度**评分：质量、速度、成本、上下文窗口、能力（任务匹配度）、可靠性。评分基于 OpenRouter 公开数据和模型规格。

#### RouterMind 可以离线使用吗？
不可以。需要联网才能与 OpenRouter API 通信。

#### 支持中文吗？
**支持。** RouterMind 完全中英双语——包括语音输入、界面和 AI 推荐。

### 故障排除

**"Invalid API Key" 错误**
- 检查设置中的 API Key 是否正确输入
- 在 [openrouter.ai/keys](https://openrouter.ai/keys) 确认密钥有效
- 确保 OpenRouter 账户有足够额度

**没有返回模型**
- 检查网络连接
- OpenRouter 可能暂时不可用
- 尝试在设置中刷新模型列表

**语音输入不工作**
- 在 iOS 设置 → RouterMind 中授予麦克风权限
- 检查设备是否处于静音模式
- 确保网络连接稳定

**应用崩溃或无响应**
- 关闭应用重新打开
- 从 App Store 更新到最新版本
- 如果问题仍然存在，请联系我们
