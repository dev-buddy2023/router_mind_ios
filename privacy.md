# Privacy Policy / 隐私政策

Last updated: June 11, 2026

## English

### Information Collection
RouterMind collects the following data to provide its service:

- **Voice input**: When you use voice features, audio is transcribed into text and sent to OpenRouter API for model recommendations. Audio data is processed locally on your device via iOS Speech Recognition and is not stored or sent to our servers.

- **API Key**: Your OpenRouter API key is stored securely in the iOS Keychain on your device. It is never transmitted to any server other than the OpenRouter API.

- **Model Data**: Model information is loaded from bundled JSON files and refreshed from OpenRouter's public API. No personal data is included in these requests.

### Data Storage
All data is stored locally on your device:
- API Key: iOS Keychain (encrypted)
- Favorites: UserDefaults (local only)
- Language preference: UserDefaults (local only)

### Third-Party Services
RouterMind communicates with:
- OpenRouter API (openrouter.ai) - to retrieve model lists and process model selection requests. See their privacy policy at https://openrouter.ai/privacy

### No Data Collection
RouterMind does not collect analytics, crash reports, or any personal information. There are no third-party SDKs for tracking or advertising.

If you have questions, contact: dev_buddy2023@163.com

## 中文

### 信息收集
RouterMind 收集以下数据以提供服务：

- **语音输入**：使用语音功能时，音频通过 iOS 语音识别转写成文字，发送给 OpenRouter API 获取模型推荐。音频数据在设备本地处理，不会存储或发送到我们的服务器。

- **API Key**：您的 OpenRouter API Key 安全存储在设备 iOS Keychain 中，仅发送给 OpenRouter API。

- **模型数据**：模型信息从本地 JSON 文件载入，并可从 OpenRouter 公开 API 刷新。这些请求不包含个人数据。

### 数据存储
所有数据存储在设备本地：
- API Key：iOS Keychain（加密存储）
- 收藏：UserDefaults（仅本地）
- 语言设置：UserDefaults（仅本地）

### 第三方服务
RouterMind 与以下服务通信：
- OpenRouter API (openrouter.ai) - 获取模型列表和处理模型选择请求。详见其隐私政策：https://openrouter.ai/privacy

### 不收集数据
RouterMind 不收集分析数据、崩溃报告或任何个人信息。没有用于追踪或广告的第三方 SDK。
